<img width="1080" height="2220" alt="image" src="https://github.com/user-attachments/assets/b98e10af-e586-4552-a920-6121d9d7e7dd" /># 💪 Fit-AI  (Frontend)

A modern frontend implementation of an AI-powered fitness platform that acts like a **personal trainer that never forgets you**.

This application allows users to track workouts, get personalized guidance, and visualize their fitness progress over time, all through a clean and responsive user interface.

---

## 🚀 Features

### 🧠 AI Fitness Assistant

* Chat-based fitness assistant for guidance and recommendations
* Personalized responses based on user activity 

### 🏋️ Workout Tracking

* Log daily workouts and exercises
* Store and manage workout history
* Structured input for consistent tracking

### 📊 Insights & Analytics

* View stats and trends from logged workouts
* Track progress over time
* Simple, intuitive data visualization

### 🛠️ Admin Dashboard *(In Progress)*

* Monitor user activity
* Track engagement with the chatbot
* View system usage insights

### 🌐 Public AI Chatbot
* Free, web-based chatbot accessible to users
* Designed for general fitness questions and guidance
* Lightweight and responsive chat interface
